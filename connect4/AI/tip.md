还需要调的参数，按优先级：

1. `UCT_C`（最关键）  
- 当前：`0.95`  
- 建议扫：`0.75 / 0.90 / 1.05 / 1.20`  
- 作用：探索 vs 利用平衡。你现在偏利用，可能对强对手不够探索。

2. rollout 候选宽度 `k = min(4, lcnt)`  
- 建议扫：`2 / 3 / 4 / 5`  
- 作用：`k` 小更贪心，`k` 大更随机。对中后盘稳定性影响明显。

3. 压缩阈值 `(MAX_NODE_POOL * 7) / 10`  
- 建议扫：`0.55 / 0.65 / 0.75`  
- 作用：复用图何时做子图压缩。太晚会内存和局部性变差，太早会复制开销大。

4. `MAX_NODE_POOL`  
- 当前：`450000`  
- 建议扫：`300000 / 400000 / 500000`  
- 作用：复用收益上限与缓存压力平衡。

5. `THINK_TIME_MS`  
- 当前：`2850`  
- 建议仅微调：`2820 / 2850`  
- 作用：超时风险控制。已接近上限，不建议再加。

6. 时间检查频率 `(iterations & 15)`  
- 建议扫：`7 / 15 / 31`  
- 作用：安全性 vs 循环开销；通常 `15` 比较稳。

如果只做最小成本高收益调参：先做 `UCT_C + k` 二维网格。
